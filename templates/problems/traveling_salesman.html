問題文

巡回セールスマン問題です．座標が重複しない0 ~ N - 1までのN個の頂点の座標xi, yiが与えられるので，訪れる順に頂点番号を出力してください．

制約
・\( 50 \leq N \leq 1000 \)
・\( 0 \leq x_{i},y_{i} \leq 1000 \)
・If \( i \neq j, (x_{i},y_{i}) \neq (x_{j},y_{j}) \)

入力
``` N x0 y0 x1 x1 . . xN-1 yN-1 ```

出力
``` v0 v1 . . vN-1 ```

スコア

v0 → v1 → ... → vN-1 → V0 と頂点を訪れた時の移動距離をスコアとする．

テスタ

TopCoderのMarathon Matchと同じです．<command>にプログラムの実行コマンド，<seed>に乱数のシードを入れてください．

$ java -jar Tester.jar -exec "<command>" -seed <seed>

その他オプション
-vis : ビジュアライズ
-save : 画像の保存
-num : 頂点番号の描画
